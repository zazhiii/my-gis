# 🌍 小型 GIS 系统

一个基于前后端分离架构的小型 GIS 系统，支持从 PostgreSQL + PostGIS 数据库中查询矢量数据并在网页端进行展示和交互。系统支持元素属性搜索、点击高亮等功能，适用于地理信息可视化与空间数据交互演示。

---

## 🚀 功能特性

- ✅ **地图可视化**：从 PostGIS 数据库查询矢量数据并在前端地图中渲染
- ✅ **属性搜索**：支持通过关键字搜索地图元素并定位
- ✅ **点击高亮**：点击地图元素后自动高亮显示并弹出属性信息
- ✅ **基础地图交互**：支持缩放、拖动、图层切换等基本功能

---

## 🛠 技术栈

| 层级     | 技术                         |
|----------|------------------------------|
| 前端     | Vue 2, Leaflet, Axios     |
| 后端     | Spring Boot, Spring Web      |
| 数据库   | PostgreSQL + PostGIS      |
| 通讯协议 | RESTful API                  |
| 数据格式 | GeoJSON                      |

---

##  环境依赖

- 已安装 Node.js 和 npm
- 已安装 Maven
- 已安装 PostgreSQL 和 PostGIS 插件
